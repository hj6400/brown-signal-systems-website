# Brown Signal Systems Website

Official website for Brown Signal Systems.

## Stack

- HTML
- CSS
- Vanilla JavaScript
- Designed for deployment on Cloudflare Pages

## Local preview

Open `index.html` in a browser.

## Deployment

Connect this repository to Cloudflare Pages. No build command is required.

- Build command: leave blank
- Build output directory: `/`
